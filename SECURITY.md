# Security Policy

## Reporting Security Issues

This is research code intended for scientific reproducibility and experimentation. While we strive for code quality, this is not production software.

If you discover a security vulnerability or issue in this research code, please report it by:

1. **Opening a GitHub Issue**: [https://github.com/gogipav14/spectral-llm/issues](https://github.com/gogipav14/spectral-llm/issues)
2. **Email**: gorgipavlov@gmail.com (for sensitive issues)

Please include:
- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

## Scope

This repository contains research code for Boolean logic synthesis experiments. It is not intended for:
- Production deployment
- Security-critical applications
- Processing of sensitive or untrusted data

For production use cases, please see the [spectralbit](https://github.com/gogipav14/spectralbit) package.

## Response

We aim to respond to security reports within:
- Critical issues: 48 hours
- Other issues: 1 week

## Supported Versions

Only the latest commit on the `main` branch is actively maintained.
